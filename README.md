# fds-brands
Brands Carousel only Css using Web Component.


### Preview:
![Alt text](https://i.ibb.co/NxcCVQy/thumbnails.png "Both sizes")

>&nbsp;
> **Config**
> Using two values "speed" and "heightimg"
> default values is 
> - speed: 120s.
> - heightimg: 150px
>&nbsp;

---
